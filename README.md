# 🌾 AgroSmart MX — Sitio Web

Sitio web público de **AgroSmart MX**, la app de gestión agrícola inteligente para agricultores mexicanos.

Este repositorio contiene únicamente la landing page, política de privacidad y términos y condiciones, publicados mediante **GitHub Pages**.

---

## 🌐 Páginas

| Página | Descripción |
|--------|-------------|
| [index.html](docs/index.html) | Landing page — presentación de la app y sus funciones |
| [privacidad.html](docs/privacidad.html) | Política de Privacidad |
| [terminos.html](docs/terminos.html) | Términos y Condiciones de Uso |

---

## 📁 Estructura

```
docs/
├── index.html          # Landing page
├── privacidad.html     # Política de Privacidad
├── terminos.html       # Términos y Condiciones
└── img/
    ├── icon.webp       # Ícono de la app
    └── icon_round.webp # Ícono redondeado
```

---

## 🚀 Activar GitHub Pages

1. Ir a **Settings** → **Pages** en este repositorio
2. En **Source**, seleccionar Branch: `main`, Folder: `/docs`
3. Guardar y esperar unos minutos a que se publique
