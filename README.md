# AgroSmartMX - Sitio Web

Este es el repositorio del sitio web para el proyecto AgroSmartMX, una plataforma dedicada a transformar la agricultura en México mediante el uso de tecnología inteligente.

## 📋 Descripción

AgroSmartMX es una solución innovadora que ayuda a los agricultores a aumentar su productividad, reducir costos y promover prácticas agrícolas sostenibles mediante el uso de:
- 🌱 Monitoreo de cultivos en tiempo real
- 💧 Gestión inteligente del agua
- 📊 Análisis predictivo de datos
- 🌤️ Pronóstico climático preciso

## 🚀 Características del Sitio Web

- Diseño responsive y moderno
- Navegación suave entre secciones
- Animaciones sutiles para mejor experiencia de usuario
- Formulario de contacto interactivo
- Optimizado para dispositivos móviles

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript vanilla

## 🌐 Visualización

Para ver el sitio web localmente:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/ariel9874/Agrosmart.git
   ```

2. Abre el archivo `index.html` en tu navegador preferido

O simplemente abre el archivo directamente desde tu explorador de archivos.

## 📝 Configuración para GitHub Pages

Para publicar el sitio en GitHub Pages:

1. Ve a la configuración del repositorio (Settings)
2. Navega a la sección "Pages"
3. En "Source", selecciona la rama `main` y la carpeta `/ (root)`
4. Guarda los cambios
5. Tu sitio estará disponible en: `https://ariel9874.github.io/Agrosmart/`
